SAFE-AI Metrics
===============

SAFE-AI Metrics is a Python package for evaluating model accuracy,
robustness, and explainability using Rank Graduation metrics.

Main metrics
------------

* RGA — Rank Graduation Accuracy
* RGR — Rank Graduation Robustness
* RGE — Rank Graduation Explainability
